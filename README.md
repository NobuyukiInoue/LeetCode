## Project for C


### Change Directory

```
cd Problems/0001_Two_Sum/Project_C
```

### Build

* Windows (Minimalist GNU for Windows)

```  
mingw32make.exe all
```

* macOS or Linux (gcc)

```
make all
```


### Run

* Windows

```
./Two_Sum.exe ../testdata.txt
```

* macOS

```
./Two_Sum_for_mac ../testdata.txt
```

* Linux

```
./Two_Sum_for_linux ../testdata.txt
```
